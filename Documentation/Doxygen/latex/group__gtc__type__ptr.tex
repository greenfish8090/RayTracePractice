\doxysubsection{Detailed Description}
Include $<$\mbox{\hyperlink{type__ptr_8hpp}{glm/gtc/type\+\_\+ptr.\+hpp}}$>$ to use the features of this extension.

Handles the interaction between pointers and vector, matrix types.

This extension defines an overloaded function, \mbox{\hyperlink{group__gtc__type__ptr_ga1c64669e1ba1160ad9386e43dc57569a}{glm\+::value\+\_\+ptr}}. It returns a pointer to the memory layout of the object. Matrix types store their values in column-\/major order.

This is useful for uploading data to matrices or copying data to buffer objects.

Example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{glm_8hpp}{glm/glm.hpp}}>}}
\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{type__ptr_8hpp}{glm/gtc/type\_ptr.hpp}}>}}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1vec}{glm::vec3}} aVector(3);}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1mat}{glm::mat4}} someMatrix(1.0);}
\DoxyCodeLine{}
\DoxyCodeLine{glUniform3fv(uniformLoc, 1, \mbox{\hyperlink{group__gtc__type__ptr_ga1c64669e1ba1160ad9386e43dc57569a}{glm::value\_ptr}}(aVector));}
\DoxyCodeLine{glUniformMatrix4fv(uniformMatrixLoc, 1, GL\_FALSE, \mbox{\hyperlink{group__gtc__type__ptr_ga1c64669e1ba1160ad9386e43dc57569a}{glm::value\_ptr}}(someMatrix));}
